# 📊 HR Analytics Dashboard

An interactive HR Analytics Dashboard built using **Power BI** to analyze employee demographics, workforce distribution, and attrition trends. The dashboard enables HR teams to monitor key workforce metrics and make data-driven decisions.

---

## 📌 Project Objective

The objective of this project is to provide HR professionals with actionable insights into employee attrition, workforce composition, and organizational trends through interactive visualizations and key performance indicators (KPIs).

---

## 🛠️ Tools & Technologies

- Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel

---

## 📈 Dashboard Features

- Total Employees
- Active Employees
- Attrition Count
- Attrition Rate
- Average Employee Age
- Employee Distribution by Gender
- Job Level-wise Employee & Attrition Analysis
- Department-wise Employee Distribution
- Business Travel Analysis
- Interactive Department Slicer

---

## 📊 Key Insights

- Monitored overall employee attrition using KPI cards.
- Compared workforce distribution across departments.
- Analyzed employee demographics by gender.
- Evaluated attrition trends across different job levels.
- Explored employee distribution based on business travel frequency.
